I. What is an Array?


An array is a data structure that consists of a collection of elements, each identified by at least one array index or key. The elements are stored in contiguous memory locations so that each element can be quickly accessed in a constant amount of time, independent of the size of the array.

-Types of Arrays
A. One-dimensional Array
A one-dimensional array is an ordered collection of related data items with the same data type. Each data item is called an element and contains a single value. All elements in an array have the same data type and are stored in a sequence in memory.
